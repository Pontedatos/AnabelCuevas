Creación del Dataframe¶

Creamos un dataframe de pandas leyendo el JSON de la base de datos

Segn la documentacin oficial de pandas, un dataframe es una estructura de datos de dos dimensiones "de tamao variable y potencialmente heterogneos".

Un JSON, segn hemos visto en la asignatura, es un tipo de familia de datos. Concretamente, JSON es una notacin de objetos en Javascript.

df = pd.read_json(url)
